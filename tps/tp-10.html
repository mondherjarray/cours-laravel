Exercice 1.2 : Test du modèle Project

3 pts
php artisan make:test ProjectTest --unit
// tests/Unit/ProjectTest.php namespace Tests\Unit; use Tests\TestCase; use App\Models\Project; use App\Models\User; use App\Models\Task; use Illuminate\Foundation\Testing\RefreshDatabase; class ProjectTest extends TestCase { use RefreshDatabase; public function test_project_belongs_to_user(): void { $user = User::factory()->create(); $project = Project::factory()->create([ 'user_id' => $user->id ]); $this->assertInstanceOf(User::class, $project->user); $this->assertEquals($user->id, $project->user->id); } public function test_project_has_many_tasks(): void { $project = Project::factory()->create(); Task::factory(3)->create(['project_id' => $project->id]); $this->assertCount(3, $project->tasks); } public function test_project_can_calculate_progress(): void { $project = Project::factory()->create(); Task::factory(2)->create([ 'project_id' => $project->id, 'status' => 'completed' ]); Task::factory(2)->create([ 'project_id' => $project->id, 'status' => 'pending' ]); // Ajoutez cette méthode dans votre modèle Project $this->assertEquals(50, $project->progress()); } }

Ajoutez la méthode progress() dans le modèle Project :

// app/Models/Project.php public function progress(): int { $total = $this->tasks->count(); if ($total === 0) return 0; $completed = $this->tasks->where('status', 'completed')->count(); return (int) (($completed / $total) * 100); }

Partie 3 : Optimisation Production (20 min)

Exercice 3.1 : Optimiser l'application

3 pts

Exécutez les commandes d'optimisation :

# Compiler les assets pour la production npm run build # Optimiser Laravel php artisan optimize # Optimiser Composer composer install --optimize-autoloader --no-dev
Note

La commande --no-dev supprime les dépendances de développement. Ne l'utilisez qu'en production.

Exercice 3.2 : Vérifier la sécurité

3 pts

Créez une checklist de sécurité pour votre fichier .env.production :

Variable Valeur Production Vérifié
APP_ENV production
APP_DEBUG false
APP_KEY Générée (32 caractères)
DB_PASSWORD Mot de passe fort
SESSION_DRIVER database ou redis
