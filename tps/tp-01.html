Partie 1 : Installation de l'environnement (30 min)

Exercice 1.1 : Installer PHP 8.2+

PHP est le langage de programmation utilisé par Laravel. Vous devez installer la version 8.2 ou supérieure.

Option A : Avec Laragon (Recommandé pour Windows)

  1. Téléchargez Laragon depuis le site officiel :
    https://laragon.org/download/
    Choisissez la version Full qui inclut PHP, MySQL, et Node.js
  2. Installez Laragon en suivant les instructions
    Cochez l'option "Ajouter au PATH" pendant l'installation
  3. Lancez Laragon et cliquez sur "Démarrer tout"

Option B : Avec XAMPP

  1. Téléchargez XAMPP depuis :
    https://www.apachefriends.org/
    Prenez la version avec PHP 8.2+
  2. Installez XAMPP dans C:\xampp
  3. Ajoutez PHP au PATH système :
    C:\xampp\php
Vérification

Ouvrez un terminal (cmd ou PowerShell) et tapez :

php -v

Vous devez voir la version de PHP (8.2 ou plus).

Exercice 1.2 : Installer Composer

Composer est le gestionnaire de dépendances PHP. Il permet d'installer Laravel et ses packages.

  1. Téléchargez l'installateur Windows depuis :
    https://getcomposer.org/download/
    Cliquez sur "Composer-Setup.exe"
  2. Exécutez l'installateur
    Quand demandé, sélectionnez le chemin vers php.exe
  3. Terminez l'installation et redémarrez votre terminal
Vérification

Dans un nouveau terminal, tapez :

composer --version

Vous devez voir : Composer version 2.x.x

Exercice 1.3 : Installer Node.js (Optionnel mais recommandé)

Node.js est utile pour compiler les fichiers CSS et JavaScript avec Vite.

  1. Téléchargez Node.js LTS depuis :
    https://nodejs.org/
  2. Installez avec les options par défaut
Vérification

Partie 2 : Création du Projet TaskFlow (30 min)

Le Projet TaskFlow

TaskFlow est une application de gestion de projets collaborative que nous allons développer tout au long de ce cours. Elle permettra de :

  • Créer des projets et des tâches
  • Organiser les tâches en tableaux Kanban
  • Assigner des tâches aux membres d'équipe
  • Suivre l'avancement des projets

Exercice 2.1 : Créer le projet Laravel

Nous allons créer un nouveau projet Laravel nommé "taskflow".

  1. Ouvrez un terminal et naviguez vers votre dossier de travail
    cd C:\laragon\www
    Ou le dossier de votre choix
  2. Créez le projet avec Composer
    composer create-project laravel/laravel taskflow

    Attendez que le téléchargement et l'installation se terminent (quelques minutes)

  3. Entrez dans le dossier du projet
    cd taskflow
En cas d'erreur
  • Vérifiez votre connexion Internet
  • Assurez-vous que PHP et Composer sont bien installés
  • Essayez de vider le cache Composer : composer clear-cache

Capture d'écran : Faites une capture de votre terminal montrant la création du projet

Exercice 2.2 : Lancer le serveur de développement

Laravel inclut un serveur de développement intégré pour tester votre application.

  1. Assurez-vous d'être dans le dossier du projet
    cd taskflow
  2. Lancez le serveur Artisan
    php artisan serve
  3. Ouvrez votre navigateur à l'adresse :
    http://localhost:8000
Résultat attendu

Vous devez voir la page d'accueil de Laravel avec le logo et le texte "Laravel".

Capture d'écran : Page d'accueil de Laravel dans votre navigateur

Exercice 2.3 : Configurer le fichier .env

Le fichier .env contient la configuration de votre application.

  1. Ouvrez le projet dans VS Code
    code .
  2. Ouvrez le fichier .env à la racine du projet
  3. Modifiez les lignes suivantes :
    APP_NAME=TaskFlow APP_URL=http://localhost:8000 DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_PORT=3306 DB_DATABASE=taskflow DB_USERNAME=root DB_PASSWORD=
Note sur la base de données

Pour ce premier TP, nous utilisons SQLite qui ne nécessite pas de configuration. Laravel 11 utilise SQLite par défaut. Si vous voulez utiliser MySQL, créez d'abord une base de données nommée "taskflow" dans phpMyAdmin.
